A minimal React application for browsing online courses. Everything is in one file for simplicity.

## Project Structure

```
src/
  App.jsx      # All components and pages in one file
  index.jsx    # React entry point
  styles.css   # All styles
```

## Install & Run

1. Install dependencies:
   ```
   npm install
   ```

2. Run the dev server:
   ```
   npm run start
   ```

3. Open `http://localhost:5173`

## Features

- 4 pages: Home, About, Courses, Contact
- Navigation bar with React Router
- Course listing with cards
- Responsive design
